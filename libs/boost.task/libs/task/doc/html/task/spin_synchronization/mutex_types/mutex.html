Class mutex 

  #include <boost/tasks/spin/mutex.hpp>

  class mutex : private boost::noncopyable
  {
  public:
      mutex();
      ~mutex();

      void lock();

bool timed_lock( system_time const& abs_time);

template< typename TimeDuration >
bool timed_lock( TimeDuration const& rel_time);

      bool try_lock();

      void unlock();

      typedef unique_lock< mutex > scoped_lock;
  };

__spinmutex_ implements the __lockableconcept_ to provide an exclusive-ownership mutex. At most one task can own the lock on a given instance of __spinmutex_ at any time. Multiple concurrent calls to _lock_, __trylock_ and _unlock_ shall be permitted.

Note

__spinmutex_ is not bound to a _scheduler_ and can only be used by tasks managed by different schedulers or code not running in a task.
