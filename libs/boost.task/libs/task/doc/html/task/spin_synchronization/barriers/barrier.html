Class barrier

#include <boost/task/spin/barrier.hpp>

class barrier
{
public:
    barrier( unsigned int initial);

    bool wait();
};

Instances of __spinbarrier_ are not copyable or movable.

barrier( unsigned int initial)

Effects:

Construct a barrier for initial tasks.

Throws:

__invalidargument_ if an error occurs.

bool wait()

Effects:

Block until initial tasks have called wait on *this. When the initial-th task calls wait, all waiting tasks are unblocked, and the barrier is reset.

Returns:

true for exactly one task from each batch of waiting tasks, false otherwise.

Throws:

__taskerror_ if an error occurs.
