Barriers

Class barrier
barrier( unsigned int initial)
bool wait()

A barrier is a simple concept. Also known as a _rendezvous_, it is a synchronization point between multiple contexts of execution. The barrier is configured for a particular number of tasks (n), and as tasks reach the barrier they must wait until all n tasks have arrived. Once the n-th task has reached the barrier, all the waiting tasks can proceed, and the barrier is reset.
