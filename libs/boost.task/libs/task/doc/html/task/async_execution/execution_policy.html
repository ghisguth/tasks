In contrast to synchronous methods asynchronous methods do not block the program flow when a time consuming operation is executed. The application continues executing the current context and when the result of the asynchronous method is required the asynchronous-completion-token can be used.

A execution-policy describes how a task gets asynchronously executed and provides a asynchronous-completion-token to manage the task.
