Synopsis

The count_down_event decrements an internal counter (set in the constructor) and all waiting fibers are blocked until the count reaches zero. 

  #include <boost/fiber/count_down_event.hpp>

  class count_down_event : private boost::noncopyable
  {
  public:
template< typename Strategy >
      explicit count_down_event( scheduler< Strategy > & sched, unsigned int initial);

      ~count_down_event();

      unsigned int initial() const;

      unsigned int current() const;

      bool is_set() const;

      void set();

      void wait();
  };

Effects:

Constructs an object of class count_down_event with initial value.

Throws:

Nothing.

~count_down_event()

Precondition:

All fibers waiting on *this have been notified by a call to set (though the respective calls to wait need not have returned).

Effects:

Destroys the object.

Throws:

Nothing.

Effects:

Returns the value the variable currently holds.

Throws:

Nothing.

bool is_set()

Effects:

Returns true if the varaible has reached zero.

Throws:

Nothing.

void set()

Effects:

Decrements the current count. If the count reaches zero and any fibers are currently blocked waiting on *this in a call to wait, unblocks those fibers. The variable remains signaled.

Throws:

Nothing.

void wait()

Effects:

Blocks the current fiber. The fiber will unblock when notified by a call to this->set() and the count of the event variable reaches zero. When the fiber is unblocked, the variable remains set.

Throws:

boost::fibers::fiber_interrupted if the wait was interrupted by a call to interrupt() on the fiber object associated with the current fiber of execution.
