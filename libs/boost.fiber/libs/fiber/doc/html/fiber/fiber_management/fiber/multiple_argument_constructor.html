template< typename Fn, typename A1, typename A2,... > fiber( Fn fn, A1 a1, A2 a2,..., std::size_t stack_size)

Preconditions:

Fn and each An must by copyable or movable.

Effects:

As if fiber( boost::bind( fn, a1, a2,...) ). Consequently, fn and each an are copied into internal storage for access by the new fiber. Second version additionaly sets the stack-size used by the fiber.

Postconditions:

*this refers to the newly created fiber.

Throws:

boost::system::system_error if system call failed.

Note:

Currently up to nine additional arguments a1 to a9 can be specified in addition to the function fn.
