Class round_robin
Synopsis

The template scheduler accepts the implementation of the scheduling algorithm as a template argument (currently round_robin is the choosen as the default). Each class destined to implement a scheduling algorithm must derive from strategy. The member active_fiber holds the current (running) fiber and master_fiber executes the scheduling logic.
