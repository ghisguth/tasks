Non-member function interruption_enabled() 

#include <boost/fiber/utility.hpp>

{
    bool interruption_enabled();
}

Returns:

true if interruption has been enabled for the current fiber, false otherwise.

Throws:

boost::fibers::fiber_error if an error occurs.
