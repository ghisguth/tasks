Function template move

boost::move —

Synopsis

template<typename T> rvalue_reference move(input_reference);

Description

This function provides a way to convert a reference into a rvalue reference in compilers with rvalue reference. For other compilers converts T & into boost::rv<T> & so that move emulation is activated.
