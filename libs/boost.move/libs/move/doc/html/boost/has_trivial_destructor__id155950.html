Struct template has_trivial_destructor_after_move

boost::has_trivial_destructor_after_move —

Synopsis

template<typename T> 
struct has_trivial_destructor_after_move {
};

Description

If this traits yields to true (has_trivial_destructor_after_move <T>value == true) menas that after moved, there is no need to call T's destructor. This optimization is used to improve containers' performance.

By default this trais is true if the type has trivial destructor, every class should specialize this trait if it wants to improve performance when inserted in containers.
